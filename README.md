# spotify_faq_chatbot
O Spotify FAQ Chatbot é uma solução de atendimento automatizada desenvolvida para responder a dúvidas frequentes de usuários do Spotify. O projeto utiliza uma arquitetura RAG (Retrieval-Augmented Generation) alimentada por Modelos de Linguagem (LLMs), garantindo privacidade, custo zero com APIs e alta precisão nas respostas.
